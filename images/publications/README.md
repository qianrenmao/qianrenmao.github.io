Place publication teaser images in this folder.

Naming convention:
- Use the publication slug when possible, for example: knowformer.png
- Reference the image in publication front matter as:
  header:
    teaser: "publications/knowformer.png"

Notes:
- The Jekyll template prepends /images/ automatically for non-URL teaser values.
- If no local image is ready yet, use "publication-placeholder.svg" temporarily.
